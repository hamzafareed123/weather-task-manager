#  CI/CD Full Stack Demo

A simple full-stack project to test **CI/CD pipeline** and **Docker** setup.

##  Tech Stack

* Frontend: React.js
* Backend: Node.js, Express, TypeScript
* Database: MongoDB
* DevOps: Docker, GitHub Actions

## Run with Docker

```bash
docker-compose up --build
```

## Run Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## CI/CD

Pipeline automatically builds and runs on every push using GitHub Actions.
